# Инструкция по генерации иконок PWA

## Шаг 1: Откройте конвертер
Откройте в браузере файл: `scripts/convert-icon.html`

## Шаг 2: Автозагрузка
Страница автоматически загрузит логотип STREAM BOOM при открытии.
Вы увидите превью иконок 192x192 и 512x512 пикселей.

## Шаг 3: Скачайте все 4 файла
Нажмите на каждую из кнопок:
1. ✅ Download icon-192.png
2. ✅ Download icon-192-maskable.png  
3. ✅ Download icon-512.png
4. ✅ Download icon-512-maskable.png

## Шаг 4: Замените файлы
Замените скачанные PNG файлы в папке `/public/`:
- Перетащите все 4 файла в `/public/`
- Подтвердите замену существующих файлов

## Шаг 5: Проверка
После замены файлов:
- Пересоберите проект
- Проверьте PWA валидатор
- Все ошибки с иконками должны исчезнуть!

## Требования PWA
✅ Иконки должны быть в формате PNG (не JPEG!)
✅ Размеры: 192x192 и 512x512 пикселей
✅ Два типа: `any` (обычные) и `maskable` (с отступами)
✅ Фон иконок: черный (#0a0a0a) под стиль приложения
